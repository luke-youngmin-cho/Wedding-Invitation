PWA 아이콘 생성 가이드

방법 1: 온라인 도구 사용

  1. Real Favicon Generator 접속
  2. assets/main.jpg 업로드
  3. Generate 클릭
  4. android-chrome-192x192.png를 pwa-192x192.png로 저장
  5. android-chrome-512x512.png를 pwa-512x512.png로 저장

방법 2: 간단한 색상 아이콘

아래 캔버스를 우클릭 → "이미지로 저장"

↑ pwa-192x192.png로 저장

↑ pwa-512x512.png로 저장
